How to Specify Polybond

In four steps

Use the Polybond product code from the digital colour library. Never use only a RAL number.

02

03

04

Write: "Powder coat finish: Polybond [code], [name], [finish]. Manufacturer: Polybond Coating Pvt. Ltd."

Light Reflectance Values

LRV for compliance

LRV measures the percentage of visible light a surface reflects. Many accessibility and building codes require a minimum LRV contrast between surfaces.

LRV Range Guide
0–20   Dark tones — absorb heat, check exterior use
20–60   Mid-tones — general architectural use
60–100   Light tones — high-contrast accessibility uses
