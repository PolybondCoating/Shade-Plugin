How to Specify Polybond

In four steps

Use this wording in your specification documents to reference Polybond colours unambiguously. All values are traceable to physical samples held at our Navi Mumbai facility.

01
Identify the colour

Use the Polybond product code (e.g. PC-4072-SM) from the digital colour library. Never use only a RAL number — RAL is not a manufacturing specification.

02
Download the TDS

Attach the Technical Data Sheet as a supporting document. This confirms film thickness, adhesion, and salt-spray ratings for your application.

03
Request a physical sample

Always approve a coated metal sample before production. Use the Sample Request on any colour page — reference the product code for traceability.

04
Reference in drawings

Write: "Powder coat finish: Polybond [PC-4072-SM], Graphite Slate, Satin Matt. Manufacturer: Polybond Coating Pvt. Ltd., Navi Mumbai." Include LRV if required.

Light Reflectance Values

LRV for compliance

LRV (Light Reflectance Value) measures the percentage of visible light a surface reflects. Values range from 0 (absolute black) to 100 (pure white). Many accessibility and building codes require a minimum LRV contrast between surfaces.

LRV Range Guide
0–20   Dark tones — absorb heat, check exterior use
20–60   Mid-tones — general architectural use
60–100   Light tones — high-contrast accessibility uses
